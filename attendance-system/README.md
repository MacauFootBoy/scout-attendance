# 澳門童軍第5旅 點名系統

## 系統概述
專為澳門童軍第5旅設計的出勤點名管理系統，支援每週集會記錄成員出席、請假、制服不整及缺席情況，並提供統計分析功能。

## 配色方案
- 主色：#1F3473 (深藍色)
- 點綴：#F2E205 (黃色)
- 輔助：#A65729 (棕色)
- 背景：#F2F2F2 (淺灰)

## 功能特色

### 1. 點名功能
- 選擇集會日期和主題
- 記錄成員狀態：✅ 出席、❌ 缺席、📝 請假、👕 制服不整
- 支援備註欄位
- 一鍵「全部出席」功能
- 批次儲存記錄

### 2. 成員管理
- 新增/編輯/刪除成員資料
- 小隊分類：幼童軍、童軍、深資童軍、領袖
- 記錄職位、電話、緊急聯絡人

### 3. 記錄查詢
- 按日期範圍篩選
- 匯出 CSV 檔案

### 4. 統計分析
- 總出勤人次統計
- 圓餅圖顯示出席比例
- 趨勢圖顯示每週出勤率
- 各小隊出勤率排名
- 個人出勤記錄

### 5. 系統設定
- 更改管理員密碼

## 啟動方式

### 方法一：使用批次檔（Windows）
```
雙擊執行 start.bat
```

### 方法二：手動啟動
```bash
cd attendance-system
npm install  (首次執行)
npm start
```

然後在瀏覽器打開：http://localhost:3001

## 預設登入
- 帳號：admin
- 密碼：scout5th

## 技術栈
- 後端：Node.js + Express + SQLite
- 前端：HTML + Bootstrap 5 + Chart.js
- 資料庫：SQLite (本地端)

## 資料儲存
所有資料會儲存在 `scouts.db` SQLite 資料庫檔案中。
